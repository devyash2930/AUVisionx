Navigating course registration can be challenging. During peak periods, heavy traffic often causes lag and errors. AUVisionX provides a reliable and efficient platform to ensure smooth and accurate course enrollment.
  1. Student
    1. View all courses:-This feature allows students to view all the available courses.
    2. View your courses:-This feature allows students to view and drop the courses they are enrolled in.
    3. Enroll to course:-Using this feature, the student can enrol into a course in which they are not already enroled in and has enough vacancy.
    4. Logout:- Logs out the student.
  2. Administrator
    1. Add course:-Allows the administrator to add a new course to the database.
    2. Delete course:-Allows the administrator to delete an existing course from the database.
